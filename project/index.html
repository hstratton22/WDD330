Project Proposal

Purpose

This project will be a test of my knowledge and skills acquired this semester. It was challenging to find a useful API that was free. I was excited for studio ghibli and LOTR but neither had much useful information in them. I considered making a matching game of prophets and their names, but I have decided to use temples instead because the church website's images have been inaccessible the past few days. I am hoping to have an image match to its name, but if that fails, I will have matching images. There are so many temples, that I have decided to limit it to United States, and I am only using ones that have the closest needed dimensions. I will only need to resize and slightly crop them. I will first create a temple's quiz similar to the superhero quiz from our readings. If I can make that work, I will progress to the memory matching game.

Audience

This project is intended for my 6-year old daughter. My teenagers may find some use as well. I think it will be a challenge for anyone since there are so many now.

Data Sources

I will be using a local json file that I will create holding the names and images.
