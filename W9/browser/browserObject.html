Browser Object notes

  • BOM
    Browser Object Model (window)
  • global
  • navigator - property that returns a reference to the Navigator object
  • Navigator object - contains information about the browser being used.
  • userAgent - property will return information about the browser and operating system being used. (deprecated)
  • Location
    The reload() method can be used to force a reload of the current page. If it’s given a parameter of true, it will force the browser to reload the page from the server, instead of using a cached page.
    The assign() method can be used to load another resource from a URL provided as a parameter.
    The replace() method is almost the same as the assign() method, except the current page will not be stored in the session history, so the user will be unable to navigate back to it using the back button.
    The toString() method returns a string containing the whole URL.
  • Window History
  • document
    document.write() will completely replace the current document.
    Placed in script tags, can inject a string.

  • Cookies
    Store information that can then be retrieved between requests. Can only be read by a web page from the same domain that set them. Limited to storing up to 4KB of data, although 20 cookies are allowed per domain. Useful for retaining state information (such as if a user is logged in) because they’re passed between the client and server on every HTTP request.
    Displayed as a text file that contain a list of name/value pairs separated by semicolons.
